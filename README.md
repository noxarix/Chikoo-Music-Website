# Chikoo Music

Chikoo Music is a modern, responsive, and sleek music streaming web application built with vanilla JavaScript, HTML, and CSS. It aims to deliver a seamless and premium listening experience.

## Features

- **Modern Glassmorphism UI:** A sleek, dark-themed user interface that looks and feels like a premium app.
- **Discover & Search:** Browse the latest trending Hindi Bollywood songs or search for your favorite tracks, artists, and albums.
- **Auto-DJ:** Endless playback based on your favorite genres (Bollywood, Pop, Lo-Fi, Punjabi, EDM, Hip-Hop).
- **Interactive Queue System:** View and interact with the "Up Next" queue in a clean side panel.
- **Live Lyrics:** Sing along with live lyrics fetched dynamically for the current playing song.
- **Playlists & Liked Songs:** Save your favorite tracks locally and create custom playlists.
- **Local Audio Support:** Drag and drop or upload your own `.mp3` or `.m4a` files to play them seamlessly.
- **Authentication & Bug Reporting:** Features built-in bug reporting that sends issues straight to the developer.

## Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **API:** JioSaavn unofficial APIs for music data and `lyrics.ovh` for song lyrics.
- **Hosting:** Deployed via Vercel / GitHub Pages.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/noxarix/Chikoo-Music-Website.git
   ```
2. Open `index.html` in your browser or run a local development server (e.g., Live Server extension in VS Code).
